# OTT Video Demo Project

## Overview
A Next.js 14 streaming platform demo application with HLS video support.

## Features
- HLS Video Streaming
- Responsive Design
- Server-Side Rendering

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
1. Clone the repository
   ```
   git clone https://github.com/erdiballikaya/ott-video-demo.git
   ```

2. Install dependencies
   ```
   npm install
   ```

3. Run development server
   ```
   npm run dev
   ```

## Project Structure
- `src/app/`: Main application routes
- `src/components/`: Reusable React components
- `src/lib/`: Utility functions and helpers

## Technologies
- Next.js 14
- React 18
- TypeScript
- HLS.js
