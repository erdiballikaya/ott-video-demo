# OTT Video Demo

This project contains a Next.js frontend and a .NET 8 backend for a minimal OTT video playback demo.

## Project Overview

### Technologies
- Frontend: Next.js (React)
- Backend: .NET 8 Minimal API
- Video Playback: HLS Streaming

### Features
- Video catalog browsing
- HLS video streaming
- Responsive design

## Local Development

### Prerequisites
- Node.js 18+
- .NET 8 SDK
- npm or yarn

### Frontend Setup
1. Navigate to frontend directory
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

### Backend Setup
1. Navigate to backend directory
2. Restore dependencies: `dotnet restore`
3. Run development server: `dotnet run`

## Deployment
- Frontend: Vercel
- Backend: Render
