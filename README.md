# OTT Video Demo

This project contains a Next.js frontend and a .NET 8 backend for a minimal OTT video playback demo.

## Project Overview

A simple Over-The-Top (OTT) video streaming application demonstrating:
- Next.js frontend with App Router
- .NET 8 Minimal API backend
- HLS video streaming
- Responsive design

## Technologies

- Frontend: Next.js 14 (App Router)
- Backend: .NET 8 Minimal API
- Video Streaming: HLS Protocol
- Deployment: Vercel (Frontend), Render (Backend)

## Local Development

### Prerequisites
- Node.js 18+
- .NET 8 SDK
- npm or yarn

### Frontend Setup
1. Navigate to frontend directory
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

### Backend Setup
1. Navigate to backend directory
2. Restore packages: `dotnet restore`
3. Run development server: `dotnet run`

## Deployment

- Frontend: Vercel
- Backend: Render
- Configured with environment-specific settings
